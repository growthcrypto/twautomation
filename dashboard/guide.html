Register Accounts in System

Go to the Accounts tab and register each account you created:

  • Enter Twitter username and password
  • Select role: Traffic or Chat
  • Select niche: Soccer, Politics, Gaming, etc.
  • Click "Register Account"

3

Upload Resources (Optional but Recommended)

Go to the Resources tab:

Add Proxies:

Enter your mobile/residential proxy details. Each account will use a different proxy.

Add API Keys:
  • 5sim API Key: For phone verification (auto-create accounts)
  • 2captcha API Key: For solving captchas automatically
  • AI API: For chat conversations (OpenAI or custom)
4

Configure Campaigns

Go to the Campaigns tab and configure:

Follow/Unfollow Campaign:
  • Max follows per day: 100
  • Delay between follows: 30-120 seconds
  • Takes breaks after 20 actions
  • Only runs 8am-10pm
Mass DM Campaign:
  • Max DMs per day: 50
  • Message template: "Hey! Check out @YourChatAccount"
  • Sends to community members
AI Chat Config:
  • Your AI API details
  • Send OF link after 12 messages
  • Auto-replies to DMs
5

Click the "Start System" button in the top right.

‚ùì Common Questions

Q: Do I need Chrome extensions?
A: No! The system controls browsers directly. No extensions needed.
Q: Where does AdsPower come in?
A: AdsPower creates isolated browser profiles (different fingerprints per account). You need it running locally or on a VPS. For Railway, you can skip it for now and test the API/dashboard.
Q: What if accounts get banned?
A: System detects bans automatically and pauses the account. You can manually create a replacement or enable auto-creation (requires API keys).
Q: How do I know it's working?
A: Dashboard shows real-time metrics (follows, DMs, leads, conversions). Check the Leads tab to see captured leads.
